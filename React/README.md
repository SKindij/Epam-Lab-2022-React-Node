# Courses Application

## Usage

Upon launching the application, you will be taken to the home page where you can login as a user or an administrator.
Depending on the role, you can either see a list of existing courses or see & change them.

**Admin:**
To create a new course, click on the "Create Course" button in the navigation bar.
Fill in the required fields for the new course, including the course title, description, authors and estimated time to complete.
Click the "Create Course" button to save the new course.
You can edit or delete existing courses by clicking on the corresponding buttons for each course.
You can add/edit/delete course authors by clicking on the corresponding buttons for each author.

**User & Admin:**
To view the details of a course, click on the View Course button.

## Dependencies

- React
- React Router
- Redux
- Redux-thunk
- Jest

## Installation

Clone the repository to your local machine.
Install dependencies by running `npm install` in your terminal.

## Running the project

In the project directory, you can run:

### `npm run dev`

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run test`

Launches the test runner in the interactive watch mode.\

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\
Your app is ready to be deployed!
